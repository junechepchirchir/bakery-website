# Chichi's Cream and Cookies

Chichi's Cream and Cookies is a responsive bakery website built using HTML, CSS, and JavaScript. The website allows users to explore the bakery, view products, browse the gallery, read blog posts, contact the bakery, and place orders online.

## Features

- Responsive design
- Multi-page website
- Navigation bar
- Dark mode toggle
- Contact form
- Online order form
- Image gallery
- Blog section

Tecnologies used are HTML 5, Css and JavaScript
